# Mersul-Trenurilor-APSII
Proiect Laborator APSII
Un sistem de gestionare a mersului trenurilor sau metroului este o platforma care ajuta la organizarea si controlul circulatiei trenurilor, astfel incat acestea sa mearga in siguranta si la timp. Sistemul primeste informatii despre pozitia trenurilor si despre starea liniilor prin senzori si echipamente montate pe infrastructura, iar intr-un centru de control aceste date sunt analizate in timp real. Pe baza lor, sistemul calculeaza automat viteza si distantele dintre trenuri pentru a evita coliziunile si pentru a reduce intarzierile. Ideea principala este ca deciziile sa fie luate cat mai mult automat, iar oamenii sa intervina doar daca apare o problema sau o situatie neprevazuta. In felul acesta se reduce riscul de eroare umana si se obtine un transport mai sigur, mai eficient si mai usor de gestionat pentru toti pasagerii.
